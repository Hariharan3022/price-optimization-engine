# Retail Price Optimization Engine

A data-driven pricing tool that models demand elasticity and finds the profit-maximizing price point.

Built by **Hariharan Gopikrishnan** — MS Business Intelligence & Analytics, Stevens Institute of Technology

## What It Does
- Models price elasticity of demand by product category
- Finds the profit-maximizing price using scipy optimization
- Side-by-side scenario comparison: Penetration / Competitor / Optimal / Premium
- Interactive scenario planner — no code needed

## Tech Stack
Python · Streamlit · SciPy · Plotly · Pandas · NumPy

## Run Locally
pip install -r requirements.txt
streamlit run app.py
